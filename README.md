# Rural Guidance App

A Flutter-based application built for rural students to provide:
- Career guidance
- Scholarships & internships info
- Social awareness articles

## Features
- Clean modern UI
- Works on Android & Web
- Offline-ready static content

## Tech Stack
- Flutter
- Dart
- Netlify (Web hosting)
